# FastApi

### FastApi란
파이썬의 빠르고 가벼운 웹 프레임 워크

### 장점
- 빠름
- 쉬움
- 직관적


### FastApi 다운
```
pip install fastapi
pip install "uvicorn[standard]"
```

### 시작 명령어
```
uvicorn main:app --reload
```
```
uvicorn {파일 이름}:{앱 이름} --reload
```

### 주소
http://127.0.0.1:8000<br>
http://127.0.0.1:8000/docs<br>
http://127.0.0.1:8000/redoc
