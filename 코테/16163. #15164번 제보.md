### 코드(복붙)
```py
def manachers(S): # N은 문자열 S의 길이
    N = len(S)
    A = [0] * N 
    r, p, result = 0, 0, 0
    for i in range(N): #
        if i <= r:
            A[i] = min(A[2 * p - i], r - i)
        while i - A[i] - 1 >= 0 and i + A[i] + 1 < N and S[i - A[i] - 1] == S[i + A[i] + 1]:
            A[i] += 1
        if r < i + A[i]:
            r = i + A[i]
            p = i
        result += (A[i]+1)//2
    return result

import sys
S = "#"+"#".join(sys.stdin.readline().strip())+"#"
print(manachers(S))
```

### 해설
Manacher는 회문의 관한 문제를 빠르게 푸는 알고리즘 이다.<br>
배열의 인덱스에 맞추어 회문의 반지름을 구한다.
