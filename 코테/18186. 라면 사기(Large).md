### 문제
https://www.acmicpc.net/problem/18186

### 코드
```py
def solution():
    n, b, c = map(int, input().split())
    arr = list(map(int, input().split()))

    if b <= c:
        return sum(arr) * b 
    
    result = 0
    for i in range(n):
        if arr[i] == 0:
            continue
        result += arr[i] * b
        if i < n - 1 and arr[i + 1] > 0:
            if i < n - 2 and arr[i + 2] > 0:
                if arr[i + 1] > arr[i + 2]:
                    d = min(arr[i], arr[i + 1] - arr[i + 2])
                    arr[i] -= d
                    arr[i + 1] -= d
                    result += d * c
                d = min(arr[i], arr[i + 1], arr[i + 2])
                arr[i + 1] -= d
                arr[i + 2] -= d
                result += d * c * 2
            else:
                d = min(arr[i], arr[i + 1])
                arr[i + 1] -= d
                result += d * c        
    return result           
            
print(solution())
```

### 해설
b가 c보다 클 경우 먼저 [i + 1] - [i + 2] 만큼  2번 사고 <br>
그 후 살 수 있는 만큼 3번 샀다.
